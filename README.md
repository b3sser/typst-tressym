# Tressym
## Render D&D 5e Character Sheets with auto-calculation

The sheet currently generated is the classic 5th edition sheet as published in the 2014 Player's Handbook, which is still the most widely used. An option to use this template for the 2024 sheet is in development, but not yet available.

### Usage
1. Make a copy of `example.typ` in `characters`.
    This can have any file name you'd like.
2. Fill the fields with your characters data
    - You may delete code lines for fields you'd like to leave blank / on their default value.
    - The default file includes info on how to use a field and what input it expects in comments.
3. Export to pdf
